# Data Structures: Doubly Linked Lists

## Doubly Linked List

<https://en.wikipedia.org/wiki/Doubly_linked_list>

A similar data structure to a singly linked list but you can traverse the collection in both directions.

```js
/*
                    Doubly Linked List


              HEAD            LENGTH = 4           TAIL
              .⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤⏤.
              │                                    │
            -----        -----       -----       -----
  null <-- │  1  │ -->  │  3  │ --> │  5  │ --> │  7  │ --> null
            -----  <--   -----  <--  -----  <--  -----  
             node         node        node        node
*/
```
